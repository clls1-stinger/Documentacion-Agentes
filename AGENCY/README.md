# AGENCY COGNITIVE ARCHITECTURE

> **Purpose**: This directory serves as the "Cerebro" (Brain) of the repository. It is the central nervous system for all autonomous agents operating within the Vega ecosystem.

## Structure

*   **0_ARCHE**: The immutable past. The origin of the project, including foundational context and historical logs.
*   **1_PRESENT**: The dynamic now. Current state, active tasks, and scratchpads.
*   **2_FUTURE**: The strategic destination. Roadmaps, backlogs, and visions.
*   **3_LEXICON**: The precise language. Definitions of terms to ensure zero ambiguity in communication.
*   **4_TEMPLATES**: The standardized forms. Templates that enforce the "infallible linguistics" required for all documentation.

## Directive

All agents must read this directory first to understand the context, state, and rules of engagement before taking any action.
